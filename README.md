# BiliBili_Video_Score_Analysis
Analysis the rank score of bilibilli daily rank board.

# Data Source
Bilibili Open Api

- https://api.bilibili.com/x/web-interface/ranking/v2?rid=0


|参数|说明|
|--|--|
|rid|排名信息分区，0为全部分区|

Response为json格式的数据


# Requirements

```
pandas
numpy
```

# Algorithm
Linear Regression
